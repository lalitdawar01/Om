# 🛡️ OM Wealth Kavach V20.7 Sentinel

NIFTY / BANKNIFTY / FINNIFTY के लिए 18-Layer Protection Bot

### Features
- **18-Kavach**: Real-time market scan हर मिनट
- **Auto Alerts**: Telegram पे तुरंत सिग्नल 
- **Sentinel Mode**: 24x7 निगरानी
- **Polling Based**: कोई Webhook झंझट नहीं

### Commands
`/start` - बॉट चालू करो
`/scan` - अभी मार्केट स्कैन
`/status` - 18-Kavach स्टेटस देखो
`/help` - सभी कमांड की लिस्ट

### Deploy on Render
1. GitHub Repo कनेक्ट करो
2. Environment में `TELEGRAM_BOT_TOKEN` डालो
3. Build Command: `pip install -r requirements.txt`
4. Start Command: `python main.py`
5. Deploy करो
